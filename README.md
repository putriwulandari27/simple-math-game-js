# 🎮 Math Adventure - Simple Math Game

Game matematika interaktif berbasis web dengan desain modern untuk melatih kecepatan berhitung (penjumlahan, pengurangan, dan perkalian).

---

## 🌟 Fitur Utama
* ➕ **Operasi Matematika Acak:** Penjumlahan & pengurangan di awal, serta perkalian otomatis muncul di ronde tinggi.
* 📈 **Sistem Ronde & Skor:** Tantangan semakin sulit seiring bertambahnya ronde.
* ❤️ **Sistem Nyawa (Lives):** Memiliki 5 nyawa per permainan.
* 🎨 **Desain Modern:** Menggunakan tampilan *Glassmorphism*, kursor interaktif, dan efek animasi latar belakang.
* 📱 **Responsive:** Nyaman dimainkan di layar HP maupun Komputer.

---

## 🛠️ Teknologi yang Digunakan
* **HTML5** - Untuk struktur halaman game.
* **CSS3** - Untuk styling *Glassmorphism*, kursor kustom, dan animasi *floating shapes*.
* **JavaScript (Vanilla)** - Untuk logika acak soal, sistem nyawa, skor, dan pindah ronde.
